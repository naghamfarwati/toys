1. mkdir : creates a new directory with the given name
.
2. ls : list files in a directory
.
3. pwd : displays the shell's current working directory.
4. cd :changes the shell's working directory to the given directory; can be a relative or absolute path.
5. rmdir: removes the directory with the given name (the directory must be empty).
6. clear: clear all previous commands' output text from the terminal.
7. rm:deletes a file.
8. touch: update the last-modified time of a file (or create an empty file).
9. cat: output the contents of a file.
10. head,tail: output the beginning or ending of a file.
11. wc: output a count of the number of characters, lines, words, etc. in a file.
12. diff: output differences between two files.
13. grep: search a file for a given string or expression.
14. curl: download from a URL and output its contents to the console.
15. git remote: Shows all the remote versions of your repository. Example: git remote origin.
16. git log: Shows a listing of commits on a branch including the corresponding details. Example: git log commit 84f241e8a0d768fb37ff7ad40e294b61a99a0abe Author: User <user@domain.com> Date: Mon May 3 09:24:05 2010 +0300 first commit.
17. git push: Pushes all the modified local objects to the remote repository and advances its branches. Example: git push origin master.
